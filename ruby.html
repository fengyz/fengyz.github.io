Interpolation: build up strings using the special syntax #{} (note: within "").

class User
  attr_accessor :name, :email

  def initialize(attributes = {})
    @name = attributes[:name]
    @email = attributes[:email]
  end

  def formatted_email
    "#{@name} <#{@email}>"    # <----- interpolation
  end
end

Ruby ampersand colon shortcut (i.e some_objects.each($:foo)

(http://stackoverflow.com/questions/1961030/ruby-ampersand-colon-shortcut)

The & calls to_proc on the object, and passes it as a block to the method. In Rails, to_proc is implemented on Symbol, so that these two calls are equivalent: 

something {|i| i.foo }
something(&:foo)

Install jupyter toc extension

Refer to http://nikgrozev.com/2015/12/27/pandas-in-jupyter-quickstart-and-useful-snippets/#toc. 

## download 
mkdir toc
cd toc
wget https://raw.githubusercontent.com/minrk/ipython_extensions/master/nbextensions/toc.js
wget https://raw.githubusercontent.com/minrk/ipython_extensions/master/nbextensions/toc.css

## install and enable
cd ..
jupyter-nbextension install --user toc
jupyter-nbextension enable toc/toc

Usage: Then we need to restart the kernel and make the first cell “Markdown” type and add the following 

<div id="toc"></div>

Or directly use the toc button on the top-right part of toolbar.

Remove the numbering of toc

Modify /root/.local/share/jupyter/nbextensions/toc/toc.css, remove the marked line 

#toc ol.toc-item li:before {
    counter-increment: item;
    content: counters(item, ".")" ";  /* <------- */
}

Keyboard shortcuts

The Jupyter Notebook has two different keyboard input modes. Edit mode allows you to type code/text into a cell and is indicated by a green cell border. Command mode binds the keyboard to notebook level actions and is indicated by a grey cell border with a blue left margin.

Mac OS X modifier keys:
  • ⌘ : Command
  • ⌃ : Control
  • ⌥ : Option
  • ⇧ : Shift
  • ↩ : Return
  • ␣ : Space
  • ⇥ : Tab
